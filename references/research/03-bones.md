# 03 骨（Bones） — L7-L9

> 呼吸と構造の層。段落のリズム、対話の処理、物語の骨格。

## L7 段落・呼吸

### 段落長

- **平均**：1,420 字
- **中央値**：162 字
- **σ**：3,297

**読解**：**中央値 162 字 ≒ 5〜7 文**が太宰の基本段落。ただし標準偏差が大きく、**極端に長い告白段落**（2,000〜5,000 字、時に 10,000 字超）が混じる。長大段落は『人間失格』はしがき・『HUMAN LOST』・『如是我聞』に顕著。

### 一段落の文数

- **既定**：5-7 文
- **短い断定段落**（1-2 文）を節目に挟む：「それだけの事である。」「嘘である。」
- **超長段落**は内的独白で使用。読点で繋いで句点で区切り、句点ごとに次の思考が湧く

### 改行頻度

- **1,000 字あたり改行**：段落が 1,420 字平均なので、約 **0.7 回／千字**
- 会話部はそれより密に改行される（台詞ごと改行）

### 一行空け

- **場面転換**で使う（時間ジャンプ・視点移動）
- **書簡・日記の差し込み**でも使う
- 感情の余白としての一行空けは稀（三点リーダで代用）

### 緩急

**基本パターン**：
1. **導入段落**（短め、150-300 字）：場面設定
2. **発展段落**（中、300-800 字）：対話や描写
3. **告白段落**（長、1,000-3,000 字）：内面吐露
4. **結節段落**（短、50-200 字）：一言でひっくり返す

『人間失格』では頁をめくるごとに「短・中・長・短」のリズムで呼吸する。

## L8 対話処理

### 会話比率

corpus 全体での会話比率は作品により大きく異なる：

| 作品 | 会話比率の印象 |
|------|----|
| 走れメロス | 中（劇的場面で増加） |
| 斜陽 | 多（母との日常会話、直治・上原との対話） |
| 人間失格 | 中（告白内の記憶の対話） |
| 女生徒 | 少（ほぼ独白） |
| ヴィヨンの妻 | 多（妻と大谷、さっちゃん視点） |
| お伽草紙 | 中（昔話の対話） |

**字数比**：corpus 全体で「」内字数は地の文の **約 15-20%**（推定、stats.json では直接計測していない）。

### 「」の扱い

- **基本形式**：「〜」。読点は「」内に入れる
- **地の文への溶解**：「」を省略して発話を地の文に流し込む（L4 自由間接話法と連動）
- **長台詞**：一つの「」内に複数文。改行せず繋げる
- **台詞の引用入れ子**：「〜と彼は言った。『〜だ』と。」

### 方言

- **津軽弁**：『津軽』『思い出』で頻出。話者は故郷の人物（父・母・乳母・兄）
- **関西弁**：なし
- **東京弁（下町）**：『ヴィヨンの妻』『斜陽』の町の人物に一部
- **標準語**：地の文・主人公内面は標準語を基本とする

**運用**：**依頼者から特に指定がなければ方言は使わない**。故郷・家族を描く場合に限り津軽弁を短く挟む。

### 話者差異

- **対話場面で語彙差を出す**：母 vs 子／夫 vs 妻／上司 vs 部下 の語彙パレット差を強く書き分ける
- 『斜陽』では、**上流の母の敬語** と **直治の破滅的口語**（「兄さんは死ぬよ」）を対比させる
- 『人間失格』の葉蔵 vs 堀木：葉蔵は内省的敬体寄り、堀木は粗野な口語

## L9 物語構造

### 基本型

作品ジャンル別に整理：

| ジャンル | 代表作 | 構造 |
|------|------|----|
| 一人称告白体 | 人間失格、HUMAN LOST | **序・告白 1・告白 2・告白 3・終**（入れ子型） |
| 一人称日記体 | 女生徒 | **朝〜夜の時間軸**（円環型）、日常の中の微細な心の動き |
| 一人称手記体 | 斜陽 | **線形＋内挿**（時系列に日記・手紙・遺書が入る） |
| 三人称ドラマ型 | 走れメロス、駆込み訴え | **古典的起承転結**。危機→追跡→決断→成就 |
| 民話・翻案 | お伽草紙、新釈諸国噺 | **原典の枠組み＋注釈**。昔話の構造を流用し、注釈で現代化 |
| 戦時随筆 | 十二月八日、苦悩の年鑑 | **日記＋社会批評**。現在時制のドキュメンタリー |

### 場面転換

- **一行空けでジャンプ**：場面切り替え・時間跳躍
- **「それから、数日経って。」** のような時間副詞で前進
- **視点転換は珍しい**：一作中で視点人物が変わるのは稀（『斜陽』の和子→直治の遺書→和子戻りが例外的）

### 時間軸操作

- **回想の入れ子**：「その時のことを思い出すたびに、私は今でも」と現在から過去へ、またすぐ現在へ戻る
- **並行展開は少ない**
- **時系列の反転**で謎を作ることは少ない（**結論先出しの告白体**が多い：「私は、人間失格。これが結論だ。」という型）

### クライマックス位置

- **早期告白型**：冒頭で「私は〇〇だ」と結論、以降は例証（人間失格、HUMAN LOST）
- **終盤凝縮型**：物語の最後で一気に落とす（走れメロス、グッド・バイ）
- **中盤の転機**：『斜陽』中盤の母の死で作品の相が変わる

**読解**：**クライマックスは「感情の頂点」であり、必ずしも「物語の解決」ではない**。解決は多くの場合、与えられない。

## 代表作 5 本の構造図

### 『人間失格』
```
はしがき（筆者視点）：3 葉の写真論
第一の手記：幼少期、道化の発見
第二の手記：中学・高校、酒・女・薬・自殺未遂
第三の手記：結婚と失敗、狂気
あとがき（筆者視点）：葉蔵のその後
```
**型**：**入れ子型告白**。外枠の第三者が内枠の告白を提示する。

### 『斜陽』
```
一：母との食卓、蛇の話
二：家を売って伊豆へ
三〜四：直治の帰還、上原との恋
五：母の死
六〜七：和子の決断、上原との子
八：直治の遺書、上原への手紙
```
**型**：**線形＋書簡挿入**。結末で和子は「ほんとうの革命」として子を産むことを決意。

### 『走れメロス』
```
導入：メロスの激怒、王への告白、人質の約束
追跡：結婚式、帰還の決意、雨と峠、裏切りの誘惑
クライマックス：磔刑場への最後の全力疾走
終結：メロスとセリヌンティウス、王の涙
```
**型**：**古典的三幕構造**。原典はシラーの『人質』。

### 『女生徒』
```
朝：起床、犬、鏡
通学：電車、少女の思考
学校：授業、友人、教師
午後：街、買い物
夜：家、食事、就寝前の独白
```
**型**：**円環的日記**。朝から夜への時間進行だが、思考は永遠に反復する。

### 『ヴィヨンの妻』
```
導入：大谷（夫）の深夜の逃避行
妻（語り手）の過去：結婚、貧困、子
大谷が盗んだ金、酒場、追跡
妻の就労、大谷との和解と再出発
結末：「人でなしでもいいじゃないの」
```
**型**：**線形的堕落譚＋倫理転換**。結末で妻が新しい倫理を発見する。

## 実行可能な運用ルール

1. **告白体を書く場合は、冒頭で結論を示してから例証を積み重ねる**（時系列逆行を恐れない）
2. **一行空けで場面転換。時間副詞（「それから」「数日後」）で時間を前に進める**
3. **段落リズムを「短・中・長・短」で循環させる**
4. **対話場面では、話者ごとに語彙パレットを切り替える**（母の敬語、若者の口語、粗野な会話）
5. **クライマックスは感情の頂点であって、解決ではない。解決を与えない結末も選択肢**
6. **方言は依頼が明示した場合のみ使う**（過剰な地方色は太宰の戦後期では避けられる）
7. **2,000 字を超える超長段落は慎重に**。内的独白の頂点でのみ使い、その前後で呼吸を戻す
8. **物語骨格は必ずしも起承転結でなくてよい**。告白体なら「宣言・例証 1・例証 2・再宣言」型が自然
