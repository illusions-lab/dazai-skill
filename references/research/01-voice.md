## L1 語彙 DNA

### 自称代名詞（頻度・比率）

| 自称 | 出現数 | 一人称内比率 |
|------|------:|---------:|
| 私 | 14,569 | 59.2% |
| 僕 | 4,840 | 19.7% |
| 自分 | 2,614 | 10.6% |
| うち | 1,007 | 4.1% |
| あたし | 838 | 3.4% |
| 我 | 372 | 1.5% |
| わたくし | 222 | 0.9% |
| 俺 | 111 | 0.5% |
| わたし | 26 | 0.1% |

**読解**：「私」が圧倒的。「僕」は中期短編（『走れメロス』冒頭は三人称だが会話は「メロス」、告白体では「僕」）と戦前作品で多い。「あたし」は女性一人称作品（『女生徒』『斜陽』の和子、『ヴィヨンの妻』のさっちゃん）で頻出。

- **運用ルール**：
  - 男性一人称なら「私」が第一選択。軽い場面・若年男性の告白体は「僕」
  - 女性一人称なら語り手年齢で使い分け：少女なら「あたし」、上流夫人なら「わたくし」、中年女なら「私」
  - **「俺」「わたし」は基本使わない**（使うときは例外を意識する）
  - 自己言及の「自分」は「私自身」の意味でも使う（二重自称の太宰流）

### オノマトペ・畳語（上位）

作品語彙としての畳語候補（*固有名詞ノイズ含むが除外せず観察する*）：

- **擬態・情感**：いよいよ 449、なかなか 349、いろいろ 267、そろそろ 182、とうとう 171、だんだん 137、もともと 108、いちいち 101、しばしば 85、ばかばか 85、うろうろ 76、わざわざ 70
- **身振り**：にやにや 89
- **外来語頻出**：ハムレット 282、ポローニヤス 154、ラプンツェル 144、ビイル 119、オフィリヤ 105、キリスト 98、マダム 96、ホレーショー 94、ウイスキイ 80、アパート 80、メロス 80、ラジオ 81、コップ 78、ベッド 78

**読解**：漢語的畳語（「頻々と」「屡々」）は使わず、和語畳語の平仮名で書く。カタカナ外来語は 1.85%（L3 参照）と低めだが、固有名詞（Shakespeare 登場人物、キリスト教概念、酒類・器物名）に偏る。**純粋擬音語・擬態語は少なく、畳語は副詞として機能している**。

### 頻出語（tf / tf-idf）から拾う太宰語彙

自然な内容語（助詞・語尾を除く）：

- **情態語**：**無い**（2,318）、**だろう**（1,056）、**知れ**〔「かも知れぬ」〕（1,275）、**思っ**（1,677）、**言葉**（1,165）、**いや**（1,693）
- **心的距離語**：**そんな**（1,964）、**あの**（2,647）、**こんな**（1,358）、**どう**（2,849）
- **呼称**：**さん**（5,783）、**お前**（786）、**あなた**（2,127）、**先生**（918）、**将軍**（515）
- **物への名指し**：**ひとり**（1,248）、**言葉**、**ところ**（2,063）、**ひと**（1,746）
- **旧字形**：**ゐる**（2,130）、**やう**（1,816）、**いふ**（1,397）、**さう**（1,055）、**言つ**（645）、**あつ**（932）、**ぢや**（498）

**読解**：**古旧仮名（ゐる／やう／いふ／さう）が 1 万例以上**。これは戦前〜戦中作品が旧字・旧仮名で執筆されたため。戦後作品では現代仮名づかい。*skill 側は現代仮名を既定とし、明示された要求がある場合のみ旧仮名で書く*。

### 固有語句（作中頻出フレーズの型）

corpus から観察される太宰固有の型（直接引用は短く）：

1. **「〜かも知れぬ」「〜かもしれない」型の逡巡終止**：「やはり人間であったかも知れない」（人間失格）
2. **自己否定の反復**：「私は、卑屈である。」「そのうえ私は、非常に弱い。」
3. **過剰敬語の効果**：斜陽冒頭「お母さまは、何事も無かったように、またひらりと一さじ、スウプをお口に流し込み…」— 「ひらりと」「お口」「すまして」の装飾重ね
4. **断定→即座の自己修正**：「醜く笑っている写真である。醜く？　けれども…」（人間失格）
5. **文末詠嘆の「のだ」**：「メロスは単純な男であった」「私には、『道化の華』があった。」

### 時代語

- **使える時代語（〜1948）**：モダン／モボ・モガ／闇市／横流し／代用食／配給／防空頭巾／疎開／兵営／徴用／英霊／玉砕／新円／MP／焼跡／斜陽族
- **使えない時代語**：戦後高度成長以降の流行語・社会現象語（カラオケ／コンビニ／SNS／バブル／リーマン／推し／エモい 等）
- **英字**：ラテン比率 0.1%。固有名詞・略号で出るが、一般名詞には基本使わない

## L2 構文・統語

### 文長分布

- **平均**：28.06 字
- **中央値**：18 字
- **標準偏差**：40.42
- **最長**：1,662 字（独白文と思われる）
- **最短**：1 字
- **総文数**：99,867

**読解**：**メジアン 18 字、平均 28 字、σ 40** という形は「短文が圧倒的多数、稀に超長文」という太宰の呼吸。短文断定でテンポを作り、要所で長い告白文を差し込む。「メロスは激怒した。必ず、かの邪智暴虐の王を除かなければならぬと決意した。」（走れメロス冒頭、10 字／23 字）は典型。**σ が平均を超える**のは、中央値から離れた外れ値が多いことを示す。

- **運用ルール**：
  - 短文連打で場面を立ち上げる。主語は最初の 1〜2 文だけ明示
  - ときどき読点で 3〜5 節を連鎖させた長文を挿入（人間失格的告白・斜陽的心理）
  - 「〜た。〜た。〜た。」の反復連打で感情の頂点を作る

### 文末処理

| スタイル | 比率 |
|--------|----:|
| 〜た（過去／完了） | 22.6% |
| 〜だ | 7.7% |
| 〜である | 6.4% |
| 体言止め | 4.8% |
| 〜です | 3.9% |
| 〜ます | 3.5% |

**読解**：**「〜た」が全体の 22%** ＝ 回想・告白の比重。**「〜である」6.4% は小川未明や志賀直哉より低く**、太宰は「である体」を避けて「〜のだ」「〜のである」の情緒的断定に寄る。敬体（〜です／〜ます）は作品内の手紙・独白や女性一人称で限定的に使用。

- **体言止め**は 4.8%（約 20 文に 1 文）。文節末の名詞で余韻を作る（「……夜ふけ。」「……ロマネスク。」）
- 倒置は fugashi 単体では直接計測できないが、会話文では頻出（「行くのか、おまえは」）

### 主語省略（推定）

- **省略率推定**：58.4%（absolutes ではなく相対値）
- **明示率**：41.6%
- サンプル文数：90,653

**読解**：*日本語文学一般の平均より若干高め*。ただし**太宰の会話文は主語を明示する**傾向がある（「メロスは激怒した」「お前は何を言っているのだ」）。独白・内的思考で主語が蒸発する。

### 段落冒頭の接続詞（実データでは少数）

| 接続詞 | 段落数 |
|------|---:|
| しかし | 6 |
| けれども | 4 |
| また | 3 |
| さて | 3 |

**読解**：**段落冒頭の接続詞は非常に少ない**。多くの段落は直接文で始まる（名詞・代名詞・人物呼称）。接続は文内で、「〜が、」「〜けれども、」と読点連結で処理される。**「しかし」「だが」を段落冒頭に置くのは、太宰らしさを損なう**。例外は論説・随筆（『如是我聞』『HUMAN LOST』）。

## L3 音韻・表記

### 文字種比率

| 種類 | 比率 |
|------|---:|
| ひらがな | 61.4% |
| 漢字 | 25.4% |
| その他（句読点・記号・空白） | 11.2% |
| カタカナ | 1.85% |
| ラテン | 0.1% |
| 数字 | 0.03% |

**読解**：**ひらがな 61% は芥川（55%前後）や志賀（50%台）より顕著に高い**。太宰はひらがなで「呼吸」する。形容詞・感情語・副詞を漢字で書かず、あえて平仮名で書く（「いや」「ちょっと」「あさ」）。**カタカナは外来語固有名詞のみ、擬音にはほぼ使わない**（「にやにや」はひらがな）。

- **運用ルール**：
  - 「有難う」より「ありがとう」、「頃」より「ころ」、「大変」より「たいへん」
  - 「云う」「為に」「所謂」は中期以降で減少、戦後は「いう」「ために」「いわゆる」中心（ただし旧仮名使用時は例外）
  - ひらがな連続で柔らかい調子を作る：「あさ、眼をさますときの気持は、面白い。」

### 句読点密度（千字あたり）

| 記号 | 密度 |
|------|---:|
| 、 | 55.3 |
| 。 | 29.2 |
| 」 | 4.9 |
| 「 | 4.9 |
| ？ | 0.84 |
| ー | 0.52 |
| （） | 0.50 ずつ |
| ！ | 0.36 |
| … | 0.19 |

**読解**：**読点 55／千字 は多い方**。短節を重ねる太宰の呼吸に由来。一文に 3〜5 個の読点が入る。**「、」と「。」の比は 1.89 ≒ 1 文あたり読点 1.89 個**。

- **ダッシュ（ー）は 0.52／千字**。会話文の長音だけでなく、思考の中断にも使う：「――わが獄中吟。」
- **！？は控えめ**（！ 0.36、？ 0.84）。*驚嘆で句読点を多用しない*（太宰は冷たい驚愕の方）
- **… は 0.19／千字 の稀少**。省略・余韻は読点と一行空けで作る

### ルビ・傍点

青空文庫データ上ではルビは除去済みだが、原文にはルビが頻出する：
- 漢字難読の訓読みルビ（「邂逅《かいこう》」「慙愧《ざんき》」）
- 方言ルビ（津軽弁）
- 外来語に漢字の当て字＋カタカナルビ（「洋袴《ズボン》」）

**skill 運用**：ルビは「読めるが耳では聴きにくい漢字」に限り使う。あまりに頻繁なルビは視覚ノイズ。

### 段落長

- **平均**：1,420.49 字
- **中央値**：162 字
- **σ**：3,297

**読解**：**中央値 162 字は一般的だが、平均 1,420 字は外れ値の存在**を示す。長大な独白段落（『HUMAN LOST』『人間失格』はしがき・手記）が平均を引き上げている。**「普通は短い段落、決定的瞬間に長大段落」**がパターン。

### 韻律・リズム

fugashi 単体では拍数計算は不十分だが、読点位置から推定：
- **読点間の平均字数**：約 17 字（= 1 節 = おおよそ 12〜14 音）
- **3 節連鎖**がモデル単位：主語節＋述部節＋副節 or 並列節
- 長文は **節リストで階段式に下る構造**（「……のだが、……のだが、……のだ。」）

## 実行可能な構文ルール（生成時に守るべき 12 項目）

1. **1 段落の平均文数は 5〜7 文、ときに 1 文だけの段落を挟む**
2. **文は短く始めて、3 文目あたりに読点 3 個以上の告白を挟む**
3. **一人称は「私」基調。軽い告白・青年的場面で「僕」。女性なら「あたし／わたくし」**
4. **三人称自己言及禁止**（「太宰は〜と考える」「太宰治は〜」は使わない）
5. **文末は「〜た／〜のだ／〜である／体言止め」を 4:3:2:1 ほどで循環させる**
6. **段落冒頭に「しかし」「だが」は置かない**。文内の「〜が、」「〜けれども、」で処理
7. **読点は 1 文に 3 個まで自然。呼吸の節目に打つ**（意味の区切りだけではない）
8. **感嘆符（！）と三点リーダ（…）は慎ましく**。驚きは短文「……そうか。」で示す
9. **固有名詞以外のカタカナを使わない**（擬態語は平仮名）
10. **漢字比率を上げすぎない**。「頃→ころ」「大変→たいへん」「云う→いう」（旧仮名設定時は別）
11. **「〜かも知れぬ／〜のだろうか」で逡巡を刻む**。断定と同じ強度で不確かさを書く
12. **自己否定→自己陶酔→再び自己否定 の対句を 20 文に 1 回くらい挿入する**

## 代表作冒頭の音の分析（4 例）

### 『人間失格』はしがき
> 私は、その男の写真を三葉、見たことがある。
- **読点 2、文末「〜ある」**：客観的静寂の始まり
- **「一葉は〜であって〜立ち〜傾け〜笑っている写真である。」** 5 節連鎖の長文。一人称だが第三者の視線

### 『斜陽』一
> 朝、食堂でスウプを一さじ、すっと吸ってお母さまが、
- **読点 3、文末「〜が、」で宙吊り**
- 「ひらりと一さじ」「すまして」「お口に流し込み」— **女性視点に乗せた華奢な動詞／副詞**

### 『走れメロス』
> メロスは激怒した。必ず、かの邪智暴虐の王を除かなければならぬと決意した。
- **10 字／23 字**：最短で動機づけ、次に目的化
- 文末「〜た」反復 2 回で劇的テンポ

### 『女生徒』
> あさ、眼をさますときの気持は、面白い。
- **平仮名書き「あさ」「眼」でひらがな先行**
- 直後に「かくれんぼのとき、押入れの真っ暗い中に、じっと、しゃがんで隠れていて、突然、…」と読点 7 個の長い比喩で継続
